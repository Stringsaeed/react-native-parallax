# react-native-parallax

Parallax ScrollView with Parallax Images and Views

## Installation

```sh
npm install react-native-parallax
```

## Usage

```js
import Parallax from "react-native-parallax";

// ...

const result = await Parallax.multiply(3, 7);
```
